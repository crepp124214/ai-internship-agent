# AI 实习求职 Agent 系统 — 设计文档

> 版本: v1.0.0 | 状态: 已完成 | 日期: 2026-04-14

---

## 一、项目定位

**定位：** 面向学生的智能求职助手

**愿景：** 通过 AI Agent 技术，帮助学生系统化准备实习申请，提升求职成功率

**核心场景：**
- 简历优化：根据目标岗位 JD 生成定向优化建议
- AI 面试：多轮对话式模拟面试，实时评分与反馈
- 岗位匹配：分析简历与岗位匹配度，量化打分

---

## 二、架构设计

```
┌─────────────────────────────────────────────────────────────┐
│                    Frontend — React SPA                      │
│   简历页 │ 岗位页 │ 面试页 │ 仪表盘 │ 设置中心               │
└─────────────────────────────────────────────────────────────┘
                              │ REST API
┌─────────────────────────────────────────────────────────────┐
│                    Backend — FastAPI + DDD                   │
│  app/api/v1/          │  domain/         │  infrastructure/  │
│  REST 端点             │  领域逻辑         │  数据库/LLM      │
└─────────────────────────────────────────────────────────────┘
                              │
┌─────────────────────────────────────────────────────────────┐
│                    Agent Runtime                             │
│  AgentExecutor │ ToolRegistry │ StateMachine │ MemoryStore   │
└─────────────────────────────────────────────────────────────┘
```

---

## 三、核心模块

### 3.1 后端结构 (`backend/`)

| 目录 | 职责 |
|------|------|
| `app/api/v1/` | RESTful API 端点（简历、岗位、面试、用户） |
| `domain/` | 领域逻辑（Agent、JD定制、面试、简历、岗位） |
| `infrastructure/` | 基础设施（数据库实体、LLM 适配器） |
| `shared/` | 共享能力（运行时、工具、错误处理） |

### 3.2 Agent Runtime (`backend/shared/runtime/`)

| 模块 | 职责 |
|------|------|
| `agent_executor.py` | ReAct 执行循环：Plan → Action → Observe → Reflect |
| `tool_registry.py` | 工具注册、发现、版本管理 |
| `state_machine.py` | 求职流程状态机 |
| `memory_store.py` | 记忆存储（会话 + 持久化） |

### 3.3 核心功能模块

| 模块 | 功能 |
|------|------|
| JD 定制 | JD 解析、匹配度分析、简历定制化重写 |
| AI 面试 | 多轮对话面试、追问、评分、复盘报告 |
| 岗位管理 | 岗位搜索、匹配度评估 |

## 四、数据流

```
用户输入 → API 端点 → 领域服务 → AgentExecutor
    ↓
ContextBuilder (会话记忆)
    ↓
Agent.plan() (分解任务，选择工具)
    ↓
循环: ToolRegistry.execute() → 观察结果 → 反思
    ↓
流式输出 (SSE) → Frontend
    ↓
结果存入 MemoryStore
```

## 五、验收标准

| 指标 | 目标 |
|------|------|
| 工具数量 | ≥ 10 个 |
| 测试覆盖率 | ≥ 80% |
| JD 匹配度评分 | 准确率 ≥ 80% |
| 面试评分合理性 | 用户满意度 ≥ 4/5 |
