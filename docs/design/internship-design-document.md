# AI 实习求职 Agent 系统 — 设计文档

> 版本: v1.0.0 | 状态: 已完成 | 日期: 2026-04-14

---

## 一、项目概述

### 1.1 项目定位

**定位：** 面向学生的智能求职助手

**愿景：** 通过 AI Agent 技术，帮助学生系统化准备实习申请，提升求职成功率

### 1.2 核心场景

| 场景 | 输入 | 输出 |
|------|------|------|
| 简历优化 | 用户简历 + 目标岗位 JD | 针对性优化的简历 + 匹配度报告 + 差异说明 |
| AI 面试 | 岗位/公司信息 + 简历上下文 | 多轮问答、即时评分、复盘报告 |
| 岗位匹配 | 用户简历 + 岗位列表 | 匹配度打分（0-100）+ 排序推荐 |

## 二、系统架构

### 2.1 整体架构

```
┌─────────────────────────────────────────────────────────────┐
│                    Frontend — React SPA                      │
│   简历页 │ 岗位页 │ 面试页 │ 仪表盘 │ 设置中心               │
└─────────────────────────────────────────────────────────────┘
                              │ REST API + SSE
┌─────────────────────────────────────────────────────────────┐
│                    Backend — FastAPI + DDD                   │
│  app/api/v1/          │  domain/         │  infrastructure/  │
│  REST 端点             │  领域逻辑         │  数据库/LLM      │
└─────────────────────────────────────────────────────────────┘
                              │
┌─────────────────────────────────────────────────────────────┐
│                    Agent Runtime                             │
│  AgentExecutor │ ToolRegistry │ StateMachine │ MemoryStore  │
└─────────────────────────────────────────────────────────────┘
```

### 2.2 数据流

```
用户输入 → API 端点 → 领域服务 → AgentExecutor
    ↓
ContextBuilder (会话记忆 + RAG 检索)
    ↓
Agent.plan() (分解任务，选择工具)
    ↓
循环: ToolRegistry.execute() → 观察结果 → 反思修正
    ↓
流式输出 (SSE) → Frontend
    ↓
结果存入 MemoryStore
```

## 三、核心模块

### 3.1 后端结构

| 目录 | 职责 |
|------|------|
| `app/api/v1/` | RESTful API 端点（简历、岗位、面试、用户、认证） |
| `domain/` | 领域逻辑（Agent、JD定制、面试、简历、岗位） |
| `infrastructure/` | 基础设施（数据库实体、LLM 适配器） |
| `shared/` | 共享能力（运行时、工具、错误处理） |

### 3.2 Agent Runtime

| 模块 | 职责 |
|------|------|
| `AgentExecutor` | ReAct 执行循环：Plan → Action → Observe → Reflect |
| `ToolRegistry` | 工具注册、发现、版本管理 |
| `StateMachine` | 求职流程状态机（投递 → 面试 → offer → 入职） |
| `MemoryStore` | 记忆存储（会话 + 持久化） |

### 3.3 核心功能

| 模块 | 功能 |
|------|------|
| JD 定制 | JD 解析、关键词提取、匹配度分析、简历定制化重写 |
| AI 面试 | 多轮对话面试、追问、评分、复盘报告生成 |
| 岗位管理 | 岗位搜索、收藏、匹配度评估 |
| 简历管理 | 简历上传、解析、优化建议 |

## 四、前端架构

### 4.1 页面结构

| 页面 | 功能 |
|------|------|
| 仪表盘 | 系统概览、快捷入口、数据统计 |
| 简历页 | 简历管理、文件上传、JD 定向优化 |
| 岗位页 | 岗位搜索、收藏、详情查看 |
| 面试页 | 面试会话、实时问答、复盘报告 |
| 设置中心 | Agent 配置、LLM 配置、用户管理 |

### 4.2 状态管理

| 方案 | 用途 |
|------|------|
| TanStack Query | 服务端状态、数据获取与缓存 |
| Zustand | 前端全局状态、UI 状态 |
| React Hook Form | 表单状态与验证 |

## 五、API 设计

### 5.1 主要端点

| 端点 | 方法 | 说明 |
|------|------|------|
| `/api/v1/auth/login` | POST | 用户登录 |
| `/api/v1/resumes/` | GET/POST | 简历列表/创建 |
| `/api/v1/resumes/{id}/optimize` | POST | 简历优化 |
| `/api/v1/jobs/` | GET/POST | 岗位列表/创建 |
| `/api/v1/jobs/{id}/match` | POST | 岗位匹配 |
| `/api/v1/interviews/sessions` | POST | 创建面试会话 |
| `/api/v1/interviews/sessions/{id}/messages` | POST | 发送面试消息 |
