# AI 实习求职工作台文案包

这份文案包用于支持 Figma 设计和后续前端实现。

## 1. 产品基础文案

### 产品名称

- 中文：AI 实习求职工作台
- 英文辅助名：AI Internship Workspace

### 产品一句话

- 让每一步求职推进都更清楚。

### 产品副文案

- 把简历、岗位、面试和投递追踪统一收进一个 AI 工作台。

## 2. 登录页文案

### 主标题

- 开始你的下一段求职推进

### 副标题

- 登录后查看简历状态、岗位匹配、面试准备和投递建议。

### 输入框

- 用户名或邮箱
- 密码

### 按钮

- 登录

### 辅助说明

- 当前版本为作品集展示版，核心展示后端能力与产品工作流。

## 9. 结果卡片短文案

### Summary 卡片

- 提炼当前简历最值得强调的核心表达。

### Match 卡片

- 对比岗位需求与简历证据，快速看到匹配度。

### Evaluation 卡片

- 用结构化反馈查看回答的强项和缺口。

### Advice 卡片

- 给当前投递阶段最值得执行的下一步动作。
